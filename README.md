#### official: 
https://deno.land
#### deno libraries: 
https://github.com/denolib/awesome-deno

### Deno installation
https://deno.land/#installation

ubuntu: 
```
curl -fsSL https://deno.land/x/install/install.sh | sudo DENO_INSTALL=/usr/local sh
```

### Run the application
```
deno run --allow-env --allow-net app.ts
```
